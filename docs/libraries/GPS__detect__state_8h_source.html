17 /*
18  GPS detection state structures. These need to be in a separate
19  header to prevent a circular dependency between AP_GPS and the
20  backend drivers.
21 
22  These structures are allocated as a single block in AP_GPS during
23  driver detection, then freed once the detection is finished. Each
24  GPS driver needs to implement a static _detect() function which uses
25  this state information to detect if the attached GPS is of the
26  specific type that it handles.
27  */
