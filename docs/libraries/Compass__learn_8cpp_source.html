8 /*
9  * this offset learning algorithm is inspired by this paper from Bill Premerlani
10  *
11  * http://gentlenav.googlecode.com/files/MagnetometerOffsetNullingRevisited.pdf
12  *
13  * The base algorithm works well, but is quite sensitive to
14  * noise. After long discussions with Bill, the following changes were
15  * made:
16  *
17  * 1) we keep a history buffer that effectively divides the mag
18  * vectors into a set of N streams. The algorithm is run on the
19  * streams separately
20  *
21  * 2) within each stream we only calculate a change when the mag
22  * vector has changed by a significant amount.
23  *
24  * This gives us the property that we learn quickly if there is no
25  * noise, but still learn correctly (and slowly) in the face of lots of
26  * noise.
27  */
