53  /*
54  Tested with: Lynxmotion Hunter Vtail 400
55  - inverted rear outward blowing motors (at a 40 degree angle)
56  - should also work with non-inverted rear outward blowing motors

58  - no yaw in front motors

64  Note: if we want the front motors to help with yaw,

71  add_motor(AP_MOTORS_MOT_3, -60, -60, 0, 4);

74  /*
75  The A-Shaped VTail is the exact same as a V-Shaped VTail, with one difference:

78  With V-Shaped VTails, the props make a V-Shape when spinning, but with

81  - Rear thrust on an A-Shaped V-Tail Quad is inward

83  Still functions the same as the V-Shaped VTail mixing below:
84  - Yaw control is entirely in the rear motors
85  - Roll is is entirely in the front motors
