48  /*
49  * A compass measurement is expected to pass through the following functions:
50  * 1. rotate_field - this rotates the measurement in-place from sensor frame
51  * to body frame
52  * 2. publish_raw_field - this provides an uncorrected point-sample for
53  * calibration libraries
54  * 3. correct_field - this corrects the measurement in-place for hard iron,
55  * soft iron, motor interference, and non-orthagonality errors
56  * 4. publish_unfiltered_field - this (optionally) provides a corrected
57  * point sample for fusion into the EKF
58  * 5. publish_filtered_field - legacy filtered magnetic field
59  *
60  * All those functions expect the mag field to be in milligauss.
61  */
